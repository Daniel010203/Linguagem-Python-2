# Faça um programa que leia na tela um número qualquer e mostre na tela a sua tabuada.
n = int(input("Digite um número para ver sua tabuuada: "))
print("Tabuada de",n,"\\n")
for i in range(1,11):
    print(n,"x",i,"=",n*1)

#Explicação do código: O código selecionado solicita que o usuário insira um número e, em seguida, 
# imprime a tabuada desse número. Aqui está uma explicação passo a passo do código: 1. `numero = int(input("Digite um número para ver sua tabuada: "))` 
#- Esta linha solicita que o usuário insira um número e o armazena na variável `número`. 
# A função `int()` é usada para converter a entrada do usuário de uma string em um inteiro. 2. `print("Tabuada de",numero,"\\n")` - 
# Esta linha imprime uma mensagem indicando que será exibida a tabuada do número digitado. O `\\n` é um caractere especial que representa uma nova linha. 
# 3. `for i in range(1,11):` - Esta linha inicia um loop que itera de 1 a 10 (inclusive). A variável de loop `i` assume os valores 1, 2, 3, ..., 10. 4. `print(numero,"x",i,"=",numero*1)` -
# Esta linha imprime cada multiplicação operação no formato "numero xi = resultado", onde `numero` é o número digitado, `i` é a variável do loop atual e `resultado` é o resultado da multiplicação de `numero` por `i`.
